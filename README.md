# EVENTLISTENER

EVENTLISTENER is a Chrome extension for monitoring live tab activity and firing one or more actions when user-defined rules are matched.

## Features

- Monitor tab, audio, click, scroll, keyboard, visibility, and DOM activity.
- Configure real conditions such as silent audio timeouts, click inactivity, scroll depth, typing bursts, and more.
- Run multiple actions when a rule fires, including alarm sound, notification, shortcut execution, stop screen share, close tab, and disarm tab.
- Manage saved defaults and live monitored tabs from the full dashboard.
- Preview alarm sounds and configure keyboard shortcuts with both manual selection and capture input.
